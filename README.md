## Intuition
Fluids are everywhere, water, honey, air, well anything that flows falls in that category. The goal was to implement a **2D Fluid** Simulation, and later expand into **3D Fluid** Simulation. All of this is based on **Jos Stam Real-Time Fluid Dynamics** Technique.
